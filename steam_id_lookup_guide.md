# Steam ID Lookup Guide

This guide helps you look up Steam IDs for all 67 players using QLStats.

## How to Find Steam IDs

1. Click the QLStats link for each player
2. If the player exists, you'll see their profile with their Steam64 ID in the URL
3. The Steam64 ID is the number after `/player/` (e.g., `76561198012345678`)
4. Copy that number to the player's YAML file in the `steamId` field

## Alternative: Community Rating Tracker

If you have access to `http://88.214.20.58/export_rating/ctf.json` or `tdm.json`, the data contains:
- `_id`: Steam64 ID
- `name`: Player name
- `rating`: TrueSkill rating

## Run the Script Locally

If you have access to the rating API, you can run the matching script locally:

```bash
python3 match_steam_ids.py
```

This will generate `steam_id_matching_report.md` with automatic matches.
